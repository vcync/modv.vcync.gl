modV

modular audio visualisation powered by JavaScript

Get Started →

Input / Output

Webcam, NDI and audio input and multi-window output, for displaying on multiple displays.

Audio Analysis

Advanced audio analysis using Meyda. modV also features automatic BPM detection and tap tempo.

Media Manager

Save and load presets with modV's built-in Media Manager. Create projects for presets, images, videos, GIFs, color palettes and more!

Versatile Controls

Assign LFOs, math expressions, MIDI inputs and audio features to controls. All assignments can be saved into presets.

Mixing and Compositing

Route, blend, pipe and control opacity per module and group. Re-arrange modules or groups via drag and drop UI.

Modular Design

modV modules support OpenGL (via WebGL) shaders, THREE.js scenes, ISF files and Canvas2D JavaScript. Using Vue.js as a base, modV has an extremely versatile plugin system.
