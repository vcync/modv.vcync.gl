# Keyboard Shortcuts

On macOS, cmd is the main shortcut key.

On Windows and Linux, Ctrl is the main shortcut key.

# Global Search

Brings up a panel which will highlight and focus any UI element in modV which matches your search term.

Ctrl/cmd + Shift + F

# Module Search

Focuses the Module Gallery's search box.

Ctrl/cmd + F
